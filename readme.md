## 1️⃣ What is the difference between var, let, and const?

//answer:


=>Var:It's a function scoped .It's can be re-declared and re-assigned.

=>Let:It's a block scoped function.It's can't be re-declared in the same scope and can be re-assigned.

=>Const:It's a block scoped function.It's can't be re-declared and can't be re-assigned.
